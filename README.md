# Mikhail Emad Alber | Portfolio 2026

A modern, industrial-minimalist portfolio website designed for a high-impact professional presence. Built with a focus on typography, structural grid layouts, and subtle interactivity.

## 🚀 Features

- **Industrial Minimalist Design**: A stark, tech-forward aesthetic using high contrast and custom typography.
- **Responsive Layout**: Fully optimized for all screen sizes, from mobile to ultra-wide displays.
- **Interactive Grid Background**: A subtle parallax grid effect that responds to mouse movement.
- **Scroll Animations**: Smooth entrance animations for sections using Intersection Observer.
- **Optimized for Deployment**: Static build ready for instant hosting on Netlify or GitHub Pages.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for better accessibility and SEO.
- **CSS3**: Vanilla CSS with modern features like CSS Variables, Grid, and Flexbox.
- **JavaScript**: Pure ES6+ JavaScript for lightweight interactivity.
- **Typography**: 
  - *Display:* Space Grotesk
  - *Body:* JetBrains Mono

## 📂 Project Structure

```text
portfolio/
├── index.html    # Main entry point
├── style.css     # Industrial minimalist styles
├── script.js     # Interactivity & animations
└── README.md     # Project documentation
```

## 🌐 Deployment

### Netlify (Recommended)
1. Drag and drop the `portfolio` folder to [Netlify Drop](https://app.netlify.com/drop).
2. Or connect this GitHub repository to Netlify for automatic CI/CD.

### GitHub Pages
1. Go to **Settings** > **Pages** in this repository.
2. Select the `main` branch and `/ (root)` folder as the source.
3. Click **Save**.
